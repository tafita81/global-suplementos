✅ O que esse comando faz (passo a passo):

1. git lfs install
   → Inicializa Git LFS no repositório

2. git lfs track "*.zip"
   → Configura para rastrear todos os arquivos ZIP via LFS

3. git add .gitattributes
   → Adiciona arquivo de configuração do LFS

4. git add .
   → Adiciona todos os arquivos (ZIPs vão via LFS)

5. git commit -m "..."
   → Cria commit com as mudanças

6. git push -f origin main
   → Envia para GitHub (ZIPs grandes serão aceitos!)
